Proyectos Realizados

Briglass Gestion

Plataforma digital desarrollada para una empresa en Francia, dedicada a la gestión integral de dossiers de bris de glace para garages automovilísticos. La solución centraliza todos los documentos y procesos, automatiza tareas repetitivas y permite el chiffrage SIDEXA, la generación de presupuestos y facturas, el seguimiento de pagos y el recouvrement.

La plataforma ofrece firma electrónica, envío de correos certificados, alta seguridad de datos y soporte especializado para profesionales del sector del vidrio automotriz.

Switch Glass

Sitio web profesional desarrollado para una empresa francesa especializada en el reemplazo de parabrisas y vitrages automovilísticos. El sitio permite a los clientes gestionar sus rendez-vous en línea, solicitar intervenciones a domicilio y recibir asistencia con aseguradoras.

El proyecto destaca por su enfoque en la experiencia de usuario, la eliminación de papeleo, la compatibilidad con todas las aseguradoras y la optimización SEO para mejorar la visibilidad online del servicio.

Hulk Pare-Brise

Proyecto web desarrollado para un garage especializado en remplacement de pare-brise en Francia. Inspirado en el modelo de Switch Glass, este sitio ofrece gestión de citas, información clara sobre servicios y contacto directo con el cliente.

Golden Code – Gestión Escolar

Plataforma de gestión escolar desarrollada inicialmente bajo el nombre SGI y posteriormente renombrada como Golden Code. El sistema permite la administración de estudiantes, gestión de notas, control de ausencias y envío de notificaciones.

Proyecto orientado a mejorar la comunicación entre secretaría, profesores y alumnos mediante una plataforma web centralizada.

Prácticas Profesionales – Deusens

Durante mis prácticas del Grado Superior DAM (24/03/2025 – 24/06/2025) en la empresa Deusens (Zaragoza), participé en un proyecto de gran envergadura para la UNESCO.

Trabajé en equipo utilizando Git con flujo basado en forks, empleando Svelte para el frontend y Laravel para el backend. También participé en reuniones con el cliente, gestión de tareas, planificación de plazos y adaptación a cambios del proyecto.
