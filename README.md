# Trabajo HTML y CSS – Aymane Belkhiri

Este proyecto consiste en un  portfolio web personal  desarrollado como trabajo final del módulo de  HTML y CSS.

El sitio presenta información sobre mi perfil profesional como  desarrollador web , incluyendo mi formación académica, habilidades técnicas, proyectos realizados, un formulario de solicitud de presupuesto y una página de contacto con datos y ubicación.

El objetivo principal del proyecto es aplicar buenas prácticas de  estructura HTML semántica ,  estilos CSS , organización de carpetas, validación de formularios y publicación del sitio en un servicio de hosting.
